# Set-AiConfig.ps1 — 仕様・使用説明書

Claude API の設定（APIキー・モデル）をユーザー環境変数に永続保存するスクリプトです。  
`MdToPdf.ps1 -AiMermaid` を使用する前に一度だけ実行してください。

---

## 動作環境

| 要件 | 詳細 |
|------|------|
| PowerShell | 5.1 以上 |
| インターネット接続 | APIキー疎通確認時に必要 |

---

## 使用方法

```powershell
.\Set-AiConfig.ps1
```

引数・パラメーターはありません。対話形式で設定を入力します。

---

## 動作フロー

### 初回実行時

```
    現在の設定: モデル=(未設定) / APIキー=(未設定)

APIキーを入力してください (Enterでスキップ): sk-ant-...
モデルを選択 [1:haiku / 2:sonnet] (Enterでスキップ): 2

[*] APIキーを確認中...
[v] 接続成功 (claude-sonnet-4-6)
[v] APIキー: 更新しました
[v] モデル: sonnet に変更しました

    設定はユーザー環境変数に保存されました（次回以降も有効です）。
```

### 再実行時（設定変更・確認）

```
    現在の設定: モデル=sonnet / APIキー=****1234

APIキーを入力してください (Enterでスキップ): （Enter）
モデルを選択 [1:haiku / 2:sonnet] (Enterでスキップ): 1

[v] モデル: haiku に変更しました

    設定はユーザー環境変数に保存されました（次回以降も有効です）。
```

### 同じ値を入力した場合

```
[~] APIキー: 変更なし（同じ値です）
[~] モデル:  変更なし（同じ値です）

    設定に変更はありませんでした。
```

---

## 設定項目

### APIキー

- 環境変数名: `ANTHROPIC_API_KEY`
- 取得先: [https://console.anthropic.com/](https://console.anthropic.com/)
- 入力時はマスク表示されます
- 設定後、Anthropic API への疎通確認を自動実行します。失敗した場合は保存されません

### モデル

- 環境変数名: `ANTHROPIC_MODEL`
- デフォルト: `sonnet`（未設定の場合も sonnet が使用されます）

| 選択肢 | モデルID | 特徴 |
|--------|----------|------|
| `1` haiku | `claude-haiku-4-5-20251001` | 高速・低コスト。シンプルなフロー図に最適 |
| `2` sonnet | `claude-sonnet-4-6` | 高品質。複雑な構成・ER図・シーケンス図に対応（推奨） |

---

## 保存先

ユーザー環境変数（`[Environment]::SetEnvironmentVariable(..., 'User')`）に保存します。

- 現在の PowerShell セッションにも即時反映されます
- システム再起動後も有効です
- 他のユーザーには見えません（システム全体には影響しません）

---

## コンソール出力の見方

| プレフィックス | 色 | 意味 |
|---|---|---|
| `[v]` | 緑 | 成功・更新 |
| `[~]` | 黄 | 変更なし（同じ値）またはスキップ |
| `[x]` | 赤 | エラー（APIキー無効・接続失敗など）|
